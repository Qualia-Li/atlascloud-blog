## Hailuo 2.3 Prompt Tips

Prompting for anime and illustration-style video follows different conventions than prompting for photorealistic content. These strategies have been tested extensively with the Hailuo 2.3 API.

### 1. Start with "Anime style" or the Specific Sub-genre

Explicitly anchoring the prompt with "anime style," "manga illustration style," "cel-shaded animation," or a specific sub-genre ("shonen action style," "Studio Ghibli aesthetic") immediately sets the model on the right track. Without this anchor, Hailuo 2.3 may default to a semi-realistic style that does not fully commit to the illustration aesthetic.

### 2. Describe Motion Using Animation Language

Instead of physics-based descriptions like "the ball bounces realistically," use animation-specific language: "the character leaps with exaggerated hang time," "speed lines trail behind the punch," "the camera snap-zooms to a close-up." Animation motion follows different rules than physics, and Hailuo 2.3 responds better to animation vocabulary.

### 3. Specify Character Design Elements

Anime characters have specific design conventions. Include details like "large expressive eyes," "stylized spiky hair," "school uniform with precise details," or "flowing cape with dramatic billowing." The more character design specifics you provide, the more polished the output.

### 4. Reference Specific Anime Aesthetics

Hailuo 2.3 responds well to references: "Studio Ghibli warmth and detail," "Makoto Shinkai sky and lighting," "Demon Slayer action choreography," "Your Name color palette." These references function as powerful stylistic anchors that align the output with established animation traditions.

### 5. Include Anime-Specific Effects

Effects that are unique to anime -- speed lines, impact frames, dramatic wind effects, sakura petals, lens flares, sparkle effects -- should be explicitly requested. These elements are part of the visual language of anime and Hailuo 2.3 renders them more convincingly than general-purpose models.

### 6. Use Emotional Descriptors

Anime is driven by emotion. Descriptions like "overwhelming determination in the character's eyes," "bittersweet farewell scene," or "comedic surprise reaction with exaggerated features" help the model capture the emotional tone that defines great anime content.
