| **Free Tier** | $0 | ~800 seconds for new accounts | One-time welcome bonus |

### Key Details

- Video output is watermarked on free and trial tiers. The monthly subscription removes watermarks.

## Quick Start: Atlas Cloud API Integration

Create a Seedance video using the Atlas Cloud API in less than 20 lines of Python.

```python
import requests
import time

API_KEY = "your-atlas-cloud-api-key"
BASE_URL = "https://api.atlascloud.ai/api/v1"

response = requests.post(
    f"{BASE_URL}/model/generateVideo",
    headers={
        "Authorization": f"Bearer {API_KEY}",
        "Content-Type": "application/json"
    },
    json={
        "model": "bytedance/seedance-v1.5-pro/text-to-video",
        "prompt": "A golden dragon soaring through aurora borealis clouds, cinematic aerial shot",
        "duration": 10,
        "resolution": "1080p"
    }
)

result = response.json()

while True:
    status = requests.get(
        f"{BASE_URL}/model/prediction/{result['request_id']}/get",
        headers={"Authorization": f"Bearer {API_KEY}"}
    ).json()
    if status["status"] == "completed":
        print(f"Video: {status['output']['video_url']}")
        break
    time.sleep(5)
```

## Money-Saving Tips

- **Use the Fast tier for iteration.** Draft and refine prompts on the $0.022/sec Fast tier. Switch to Pro only for final renders where maximum quality matters.
- **Keep durations short.** A 5-second clip costs half as much as a 10-second clip. Many social media formats perform better with shorter, punchier clips anyway.
- **Batch similar prompts.** Group related generations together to minimize context-switching overhead and maximize productive output per dollar.
- **Leverage free tiers for testing.** Use Jimeng's daily login credits or Dreamina's free tokens to test prompt ideas before committing API budget.
- **Choose the right aspect ratio upfront.** Regenerating a clip because the aspect ratio was wrong doubles your cost. Plan the output format before generating.
- **Compare models for your use case.** Veo 3.1 at $0.03/sec may be more cost-effective for certain styles. Test across models using Atlas Cloud's multi-model API before committing to a single pipeline.
